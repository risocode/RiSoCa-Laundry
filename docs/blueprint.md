# **App Name**: RKR Laundry

## Core Features:

- Order Creation: Guided order creation flow with step-by-step selection.
- Real-Time Order Tracking: Live status tracker reflecting operational workflow with animated progress bar and timestamped status logs.
- Service Selection & Dynamic Pricing: Smart pricing engine automatically computes total cost based on service selections and distance inputs, incorporating distance-based surcharges and optional service add-ons.
- Address Management: User profile includes multiple saved addresses with integrated map picker for precise pin-location for pickup or delivery.
- Automated Notifications: In-app dashboard notifications for every status update with optional SMS or email notifications and retry logic.
- Pricing Logic Evaluator: Tool which is a decision engine that identifies the correct pricing category based on user selections, validates combinations, flags invalid service choices, and outputs the computed price.
- Loyalty Points System: Points earned per completed order that customers can redeem for discounts or free services, trackable through profile dashboard.

## Style Guidelines:

- Primary: Deep Purple (#673AB7) — conveys reliability and cleanliness.
- Background: Light Lavender (#EDE7F6) — soft, clean, minimal visual weight.
- Accent: Soft Amber (#FFB300) — ideal for buttons, highlights, and interactive cues.
- Font Family: PT Sans (for both headers and body text).
- Card-based layout for sections such as orders, tracking, and pricing breakdown.
- Line-based icons for services and statuses (wash, dry, fold, pickup, delivery).
- Shadowed containers with rounded corners for depth; clear hierarchy: bold headers, secondary captions, and structured spacing.
- Smooth transitions between order stages and animated progress bar for order status.
- Subtle hover effects on actionable items (CTA buttons, cards, toggles) and auto-refresh for status updates with fade-in effect.
